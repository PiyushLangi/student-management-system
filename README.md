# 🎓 Student Management System

A Python-based Student Management System that allows users to manage student records efficiently using a modular programming approach and JSON file storage.

## 🚀 Features

- Create new class and student records  
- Search students by multiple criteria  
- Update student information  
- Delete class or student records  
- Menu-driven console interface  
- Modular and clean code structure  

## 🛠️ Tech Stack

- Python  
- JSON (for data storage)  
- Modular Programming
